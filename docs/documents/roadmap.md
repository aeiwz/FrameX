# Roadmap

## 2026 Milestones

- API contracts and product definition
- storage layer evolution (Arrow-backed partitioning)
- local execution engine and transport optimization
- operator expansion (groupby, join, window, sort)
- packaging and beta hardening

## Active Focus Areas

- clarify compatibility boundaries with Pandas
- improve lazy planning and optimization depth
- strengthen process/thread scheduling heuristics
- expand benchmarks and regression coverage

## Stability and Versioning

FrameX is pre-1.0. Expect interface growth and targeted behavior changes as performance and compatibility converge.

For production adoption, pin versions and validate key workflows against your reference outputs.
